# ABSA
Aspect-based sentiment analysis on 47k Russian restaurant reviews (food/service/interior). Fine-tuned ruBERT-base, ruBERT-tiny and mBERT; best result — ruBERT-base test accuracy = 0.795. Hyperparameter tuning via Optuna, early stopping, ablation study.
